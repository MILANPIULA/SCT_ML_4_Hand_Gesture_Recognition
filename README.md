# ✋ Hand Gesture Recognition using Convolutional Neural Network (CNN)

## 📌 Project Overview

This project focuses on building a Hand Gesture Recognition system using a Convolutional Neural Network (CNN). The model is trained to recognize three different hand gestures: **Rock**, **Paper**, and **Scissors**. The project demonstrates the use of Deep Learning techniques for image classification and computer vision applications.

---

## 🎯 Objective

The objective of this project is to develop a CNN-based image classification model capable of accurately identifying different hand gestures from images. This project was completed as part of the **SkillCraft Technology Machine Learning Internship – Task 4**.

---

## 📂 Dataset

The project uses the **Rock-Paper-Scissors Dataset** provided by TensorFlow.

The dataset contains images of three gesture classes:

- ✊ Rock
- ✋ Paper
- ✌️ Scissors

The images are resized and normalized before training.

## 🧠 Deep Learning Model

A Convolutional Neural Network (CNN) was built using the following layers:

- Conv2D
- MaxPooling2D
- Flatten
- Dense Layers
- Softmax Output Layer

The model was compiled using:

- **Optimizer:** Adam
- **Loss Function:** Categorical Crossentropy
- **Evaluation Metric:** Accuracy

---

## 🔄 Project Workflow

1. Upload and extract the dataset.
2. Load images using ImageDataGenerator.
3. Normalize image pixel values.
4. Split the dataset into training and validation sets.
5. Build the CNN model.
6. Train the model for multiple epochs.
7. Evaluate model performance.
8. Save the trained model.
9. Visualize training and validation accuracy/loss.

---

## 📊 Results

The CNN model successfully learned to classify hand gestures with high training and validation accuracy. The performance graphs show effective learning with reduced loss over the training epochs.
